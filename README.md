# 🎯 Zuma Game

Classic ball matching game built with HTML5, CSS3, and JavaScript.

## 🎮 How to Play

1. **Move mouse** to aim the cannon
2. **Left click** to shoot balls
3. **Right click** to swap current ball (with cooldown)
4. Match **3 or more balls** of the same color to destroy them
5. Prevent balls from reaching the end of the path!

## 🎯 Features

- Classic Zuma gameplay with spiral path
- 6 different ball colors
- Level system with increasing difficulty
- Combo system for consecutive matches
- Particle effects for explosions
- Swap ball ability (right-click)
- 3 lives system
- Score tracking

## 🎮 Controls

| Action | Control |
|--------|---------|
| Aim | Mouse Move |
| Shoot | Left Click |
| Swap Ball | Right Click |
| Pause | Space / Esc |

## 📊 Scoring

- Base match (3 balls): 30 points
- Each additional ball: +10 points
- Combo multiplier: up to 5x
- Chain reactions: bonus points

## 🛠️ Tech Stack

- HTML5 Canvas
- Vanilla JavaScript (ES6+)
- CSS3 with animations
- No external dependencies
